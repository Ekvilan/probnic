PHP и JavaScript

Языки программирования PHP и JavaScript позволяют сделать сайт динамичным, то есть реагирующим на действия пользователя. Например, можно сделать красивую и выпадающую менюшку

Виды скриптов

Для этого пишутся скрипты (англ.script - "сценарий") - программы, позволяющие реагирвоать на действия пользователя. Скрипты бывают двух видов:

  • те, которые выполняются на сервере, а результат их выполнения приходит в браузер к пользователю уже в готовом виде. Это скрипты, написанные на языке PHP. на нем пишутся СMS-ки - системы управления контентом
  • те, которые выполняются прямо в браузере пользователя. Это скрипты JavaScript. Они чаще всего используются для, того чтобы сделать страницу более удобной и красивой.
